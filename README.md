# PM1 React App

โปรเจกต์นี้เป็นระบบจัดการงาน (TPSMA Project Management) ตัวอย่าง  
- สร้างด้วย React
- ใช้ mock data สำหรับทดสอบ
- มีระบบ Login, Dashboard, เปลี่ยนสถานะงาน, และ Logout
- รองรับ Responsive ทุกอุปกรณ์
